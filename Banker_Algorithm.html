银行家算法

颜色说明

Need = Max − Allocation
安全性算法：找到一个进程序列，使每个进程都能在 Work（可用资源）满足其 Need 后执行完毕。

Need ≤ Work（条件满足）   Need > Work（条件不满足）
Alloc 归还 / Work 减少   进程已完成行
